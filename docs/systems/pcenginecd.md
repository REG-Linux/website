## Overview

Introduced in 1988 by NEC, the PC Engine CD was a console system.

## Technical specifications

- Manufacturer: NEC
- Release year: 1988
- Hardware type: console

## Supported ROM extensions

pce, cue, ccd, iso, img, chd
