## Overview

Introduced in 1981 by NEC, the PC-8800 was a computer system. Its platform tag is `pc88` for proper filtering.

## Technical specifications

- CPU: Zilog Z80A clocked at 4 MHz with multiple I/O ports for keyboards and disk controllers.
- Memory: 64 KB RAM (expandable via memory boards) plus 32 KB ROM housing BASIC.
- Display: NEC µPD7220-driven graphics offering 640×400 resolution with eight colors and hardware drawing primitives.
- Sound: Yamaha YM2203 (3-channel FM + 3 PSG) providing rich music/sound capabilities.

## Supported ROM extensions

cmt, d88, u88, m3u
