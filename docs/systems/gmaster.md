## Overview

Introduced in 1990 by Hartung, the Game Master was a portable system.

## Technical specifications

- Manufacturer: Hartung
- Release year: 1990
- Hardware type: portable

## Supported ROM extensions

bin, zip, 7z

## Emulators

- **mame** (libretro)
- **mame** (mame)

## Notes
