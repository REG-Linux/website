## Overview

Introduced in 1990 by Nintendo, the Super Nintendo Entertainment System was a console system. It is grouped with snes titles in EmulationStation.

## Technical specifications

- Manufacturer: Nintendo
- Release year: 1990
- Hardware type: console
- EmulationStation group: snes

## Supported ROM extensions

smc, fig, sfc, gd3, gd7, dx2, bsx, swc, zip, 7z

## Emulators

- **bsnes** (libretro) – Requires BR2_PACKAGE_LIBRETRO_BSNES
- **bsnes_jg** (libretro) – Requires BR2_PACKAGE_LIBRETRO_BSNES_JG
