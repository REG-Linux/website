## Overview

Introduced in 1995 by Nintendo, the Satellaview was a console system. It is grouped with snes titles in EmulationStation.

## Technical specifications

- Manufacturer: Nintendo
- Release year: 1995
- Hardware type: console
- EmulationStation group: snes

## Supported ROM extensions

bs, smc, sfc, zip, 7z
