# Astrocade

## Overview

Introduced in 1978 by Bally, the Astrocade was a console system.

## Technical specifications

- Manufacturer: Bally
- Release year: 1978
- Hardware type: console

## Supported ROM extensions

bin, zip, 7z

## Emulators

- **mame** (libretro)
- **mame** (mame)

## Notes

Requires MAME BIOS file astrocde.zip
