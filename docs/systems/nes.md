## Overview

Introduced in 1983 by Nintendo, the Nintendo Entertainment System was a console system. It is grouped with nes titles in EmulationStation.

## Technical specifications

- Manufacturer: Nintendo
- Release year: 1983
- Hardware type: console
- EmulationStation group: nes

## Supported ROM extensions
