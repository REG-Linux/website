## Overview

Introduced in 1978 by APF Electronics, the M-1000 was a console system.

## Technical specifications

- Manufacturer: APF Electronics
- Release year: 1978
- Hardware type: console

## Supported ROM extensions

bin, zip, 7z

## Emulators

- **mame** (libretro)
- **mame** (mame)

## Notes

Requires MAME BIOS file apfm1000.zip
