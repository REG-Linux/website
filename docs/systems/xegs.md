## Overview

Introduced in 1987 by Atari, the Atari XE Game System was a computer system. It is grouped with atari8bit titles in EmulationStation.

## Technical specifications

- Manufacturer: Atari
- Release year: 1987
- Hardware type: computer
- EmulationStation group: atari8bit

## Supported ROM extensions

atr, dsk, xfd, bin, rom, car, zip, 7z

## Emulators

- **mame** (libretro)
- **mame** (mame)

## Notes

Requires MAME BIOS file xegs.zip
Software list mode recommended
