## Overview

Introduced in 1986 by Nintendo, the Family Computer Disk System was a console system. It is grouped with nes titles in EmulationStation.

## Technical specifications

- Manufacturer: Nintendo
- Release year: 1986
- Hardware type: console
- EmulationStation group: nes
