# Odyssey2

## Overview

Introduced in 1978 by Magnavox - Philips, the Odyssey2 was a console system.

## Technical specifications

- CPU: Intel 8048 microcontroller running at 1.79 MHz with built-in ROM for the console logic.
- Memory: 64 bytes of scratchpad RAM plus additional video latches for character data stored in the cartridge.
- Display: 160Ã—200 fixed-resolution color graphics with a 16-color palette and hardware block sprites.
- Sound: Native single channel tone generator with noise tones and optional external speaker volume control.
