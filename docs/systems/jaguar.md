# Jaguar

## Overview

Introduced in 1993 by Atari, the Jaguar was a console system. It is grouped with jaguar titles in EmulationStation.

## Technical specifications

- Manufacturer: Atari
- Release year: 1993
- Hardware type: console
- EmulationStation group: jaguar

## Supported ROM extensions

cue, j64, jag, cof, abs, cdi, rom, zip, 7z

## Emulators

- **bigpemu** (bigpemu) – Requires BR2_PACKAGE_BIGPEMU
- **virtualjaguar** (libretro) – Requires BR2_PACKAGE_LIBRETRO_VIRTUALJAGUAR | Incompatible extensions: zip, 7z
