## Overview

Introduced in 1980 by Nintendo, the Game and Watch was a portable system. It is grouped with lcdgames titles in EmulationStation.

## Technical specifications

- Manufacturer: Nintendo
- Release year: 1980
- Hardware type: portable
- EmulationStation group: lcdgames

## Supported ROM extensions

mgw, zip, 7z

## Emulators

- **gw** (libretro)
- **mame** (libretro)
- **mame** (mame)
