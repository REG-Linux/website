# BBC Micro

## Overview

Introduced in 1981 by Acorn Computers, the BBC Micro was a computer system.

## Technical specifications

- Manufacturer: Acorn Computers
- Release year: 1981
- Hardware type: computer

## Supported ROM extensions

mfi, dfi, hfe, mfm, td0, imd, d77, d88, 1dd, cqm, cqi, dsk, ima, img, ufi, 360, ipf, ssd, bbc, dsd, adf, ads, adm, adl, fsd, wav, tap, bin, zip, 7z

## Emulators

- **mame** (libretro) – Requires BR2_PACKAGE_LIBRETRO_MAME, BR2_PACKAGE_HAS_LIBRETRO_MAME
- **mame** (mame) – Requires BR2_PACKAGE_MAME, BR2_PACKAGE_HAS_MAME
