Install notes

Flash {{ device_title }} and boot

Follow the workflow below; the official board page contains any extra quirks for the {{ soc_list | join: ', ' }} platform.

  1. Grab the image. Download the latest {{ device_title }} build from the downloads page or the official board entry.
  2. Flash to storage. Use balenaEtcher, Raspberry Pi Imager, or `dd` to write the image to an SD card or SSD. Make sure to verify the checksum before booting.
  3. First boot setup. Insert the media into your {{ device_title }}, power on, and let REG Linux expand the filesystem. Pair controllers inside REG-Station once prompted.
  4. Troubleshoot with rescue tools. REG Linux ships with an immutable base and rescue partitionâ€”if something breaks, reboot into rescue or reflash without touching your ROMs.
