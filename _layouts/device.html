Installation

  1. Download. Get the latest {{ device_title }} image from the downloads page{% if device_board_url %} or the wiki{% endif %}.
  2. Flash. Write the image to an SD card or SSD using balenaEtcher, Raspberry Pi Imager, or dd.
  3. Boot. Insert the media, power on. REG Linux expands the filesystem on first boot. Pair your controllers when prompted.
  4. Rescue. If something goes wrong, reboot into rescue mode — your games are always safe on a separate partition.
