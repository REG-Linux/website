## Install REG Linux

### Installation

Download the current RK3566-X55 REG Linux image for this device from the REG Linux download page, then follow the steps in the Installation guide.

Once you have burned the image to an SD card, you will need to modify the file extlinux/extlinux.conf on the FAT partition, and change x55 to x35s on the FDT line.
