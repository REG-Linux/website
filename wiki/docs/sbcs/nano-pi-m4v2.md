## Installation notes

1. Grab the image. Download the latest Nano Pi M4v2 build from the downloads page.

2. Flash to storage. Use balenaEtcher, Raspberry Pi Imager, or `dd` to write the image to an SD card or eMMC. Make sure to verify the checksum before booting.

3. First boot setup. Insert the media into your Nano Pi M4v2, power on, and let REG Linux expand the filesystem. Pair controllers inside EmulationStation once prompted.
