# Nano Pi K2

Nano Pi K2 pairs the Amlogic S905(X) SoC with the REG Linux stack for a polished retro console.

## Installation notes

1. Grab the image. Download the latest Nano Pi K2 build from the downloads page.

2. Flash to storage. Use balenaEtcher, Raspberry Pi Imager, or `dd` to write the image to an SD card or eMMC. Make sure to verify the checksum before booting.

3. First boot setup. Insert the media into your Nano Pi K2, power on, and let REG Linux expand the filesystem. Pair controllers inside EmulationStation once prompted.
