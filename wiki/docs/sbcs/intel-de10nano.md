## Installation notes

1. Grab the image. Download the latest Unknown DE10_NANO build from the downloads page.

2. Flash to storage. Use balenaEtcher, Raspberry Pi Imager, or `dd` to write the image to an SD card or eMMC. Make sure to verify the checksum before booting.

3. First boot setup. Insert the media into your Unknown DE10_NANO, power on, and let REG Linux expand the filesystem. Pair controllers inside EmulationStation once prompted.
