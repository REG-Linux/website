## Overview

Introduced in 1983 by Acorn Computers, the Electron was a computer system.

## Technical specifications

- Manufacturer: Acorn Computers
- Release year: 1983
- Hardware type: computer

## Supported ROM extensions

wav, csw, uef, mfi, dfi, hfe, mfm, td0, imd, d77, d88, 1dd, cqm, cqi, dsk, ssd, bbc, img, dsd, adf, ads, adm, adl, rom, bin, zip, 7z

## Emulators

- **mame** (libretro)
- **mame** (mame)

## Notes

Requires MAME BIOS files electron.zip, electron64.zip, electron_plus1.zip, electron_plus3.zip
Using software list mode (cassette) is recommended.
Tapes should auto-load, use the MAME menu to stop/play/rewind if needed.
